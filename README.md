# AI-IMPACT-AMONG-STUDENTS--POWERBI
Power BI dashboard analysing AI usage among students
Project Overview

This project analyses the impact of AI tools such as ChatGPT, Copilot, Gemini, Grammarly, and Notion AI on students' daily life and academic performance.

The dashboard provides insights into usage patterns, demographics, and performance impact.

🎯 Objective
Analyse how students use AI tools
Understand impact on academic performance
Identify usage trends across gender, age, and location
📂 Dataset Features
AI Tool Used
Age
Gender
City
Daily Usage Hours
Impact on Grades
Purpose (Coding, Writing, Research, etc.)
📊 Key Insights
Students spend ~3 hours daily on AI tools
Coding is the most common use case
ChatGPT shows balanced academic outcomes
Gemini has the highest overall usage
AI adoption is consistent across genders
Usage is higher in urban regions
